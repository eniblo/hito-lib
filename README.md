# hito-lib

## components

###Select###

inspired by "ionic select".

###Toast###

modified Onsen UI notification component.

```
hito.toast.show({
  title:"Complete!",
  message:"Please add next",
  time:1000
});
```

###Loading###

Show/Hide modal loading view. 

```
hito.loading.show({
  title:"Loading",
  message:"Please wait",
  icon:"fa-home"
})

hito.loading.hide();
```
